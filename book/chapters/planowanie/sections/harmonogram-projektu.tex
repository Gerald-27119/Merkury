\section{Harmonogram projektu}
\label{sec:harmonogram-projektu}

W poniższym harmonogramie przedstawiono plan prac nad poszczególnymi częściami projektu.

\begin{itemize}
    \item \textbf{Lipiec - wrzesień 2024}
    \begin{itemize}
        \item Przygotowanie środowiska deweloperskiego.
        \item Rozpoczęcie prac nad logowaniem i rejestracją użytkownika.
    \end{itemize}

    \item \textbf{Październik 2024 - styczeń 2025}
    \begin{itemize}
        \item Wybór tematu projektu.
        \item Analiza grupy docelowej.
        \item Wstępne opracowanie wymagań.
        \item Pełne rozpoczęcie prac deweloperskich.
    \end{itemize}

    \item \textbf{Luty 2025 - wrzesień 2025}
    \begin{itemize}
        \item Prace deweloperskie nad backendem i frontendem aplikacji.
        \item Iteracyjne rozszerzanie funkcjonalności i refaktoryzacja kodu.
    \end{itemize}

    \item \textbf{Październik 2025 - styczeń 2026}
    \begin{itemize}
        \item Finalizacja prac deweloperskich.
        \item Implementacja testów automatycznych głównych funkcjonalności.
        \item Przygotowanie i ukończenie części tekstowej pracy inżynierskiej.
    \end{itemize}
\end{itemize}
